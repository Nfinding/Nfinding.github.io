我的Mysql入门

发表于 2017-02-28 | 分类于 Mysql入门

当我遇到NodeJs之后，好像通过这个’门缝’渐渐对后台有了了解，然后就有了试试本地数据库的想法，然后就在众多数据库（关系型数据库:oracle、miscrosoft Sql Server、mysql、postgresql（以稳定性著称）非关系数据库:redis数据库（内存数据库））中选择了mysql。

MySQL数据库

MySQL数据管理系统由瑞典的MySQLDB公司开发，但是几经辗转，现在已经是Oracle的产品了。

开源、免费、轻量
MySQL具有跨平台的特性，不仅可以在Windows平台上使用，还可以在Unix、Linux和Mac OSX平台上使用。
相对其它数据库而言，MySQL的使用也更加方便、快捷而且MySQL免费所以运营成本低，因此很多公司都在用MySQL。
Oracle收购过来之后，发行了两个版本：一个是企业版，公司买了之后，Oracle公司可以提供一些服务，例如数据恢复，或者说存储数据出现了问题
一个社区版，也就是免费开原版
随便用，免费滴，但是出了问题，没人管，自己去解决

通过在cmd命令台中输入path查看是否有mysql的bin目录

二、MySQL目录结构

bin：可执行文件所在目录
data：数据文件所在目录
Docs：文档目录
share：存放字符集等信息
my.ini：MySQL数据库使用的配置文件

三、MySQL的使用

启动 和 停止MySQL服务

通过Windows服务管理器启动MySQL服务
通过Windows的运行，输入services.msc找到MySQL服务
通过DOS命令启动MySQL服务
net stop mysql 停止MySQL服务
net start mysql 开启MySQL服务

登录MySQL数据库

使用相关命令登录 修改密码:
打开命令台：
mysql -h localhost -P 3306 -u root -p
-h：主机名
-P：端口
-u：用户名
-p：密码
这种方式一般用来连接远程数据库服务器
mysql默认连接localhost和3306，所以可以省略-h和-P
mysql -u root -p
这种方式一般用来连接本机，可以省略-h和-P，默认就是localhost和3306

四、数据库基础指令（注意：一定要在语句的末尾加分号; db_name数据库名称）

1、创建数据库 create database *;
2、查看数据库
查看当前数据库服务器中所有的数据库：show databases;
显示数据库创建语句
show create database db_name;
3、查看mydb1数据库的创建语句
show create database mydb1;
4、删除数据库
drop create database db_name;
5、选择数据库
use db_name;
查看当前使用的数据库
select database();

五、数据表的基本操作

1、创建数据表
（datatype:数据类型<选择数据类型：首选足够用，其次尽量小>，具体数据类型不在本文做具体说明）
create table table_name
(
field1 datatype,
field2 datatype,
field3 datatype,
)
2、查看数据表
查看当前数据库中的所有表
show tables;
查看表结构
desc table_name;
查看建表语句
show create table table_name;
3、修改数据表（colum 代表你新增加字段的名字）
增加列
ALTER TABLE table_name ADD colum datatype;
修改列
ALTER TABLE table_name MODIFY colum datatype;
删除列
ALTER TABLE table_name DROP colum;
修改表名
rename TABLE table_name to new_table_name;
修改列名
ALTER TABLE table_name change colum_name new_colum_name datatype;
