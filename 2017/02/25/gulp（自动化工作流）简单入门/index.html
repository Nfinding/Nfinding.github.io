gulp（自动化工作流）简单入门

发表于 2017-02-25 | 分类于 工具

gulp（http://gulpjs.com/）

在我看来工程的开始，环境是最基础、最前提
nodejs npm是使用gulp的前提

1.安装

安装全局的gulp-cli（gulp命令行工具） npm install gulp-cli -g
在项目中安装gulp npm install gulp –save-dev
gulp中文文档（https://github.com/lisposter/gulp-docs-zh-cn）

2.常用API

gulp.src(）找文件
gulp.dest() copy到哪儿
gulp.task() 声明一个任务
gulp.watch() 监视文件

3.gulp常用插件已经browser-sync的使用

gulp-less
gulp-cssnano
gulp-concat
gulp-uglify
